## Note on P1 → P2 → P3 as one cycle
P1, P2, and P3 are not three independent principles — they are three phases of one compounding loop: P1 produces the System Design → P2 tests it against Acceptance Criteria (A.C.) → P3 traces the outcome back to specific design decisions and updates the System Design → back to P1. Each cycle makes the next design better. The ILE (Integrated Learning Environment) is the digital environment built on Cursor to run this cycle.
